## 📍 How to use?

### CLIP Repository
1. Clone CLIP Repository
2. Get dependency
```
    flutter pub get
```

### CLIP_AI Repository
1. Clone CLIP_AI Repository
2. Install
   
```
    pip install uvicorn
    pip install python-multipart
    pip install ultralytics
```

3. Execute
```
    uvicorn predict:app —reload
```
